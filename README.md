# Instruções

-   Copiar o código para `main.py`

-   Rodar o comando `pytest -vvs`

-   **NÃO** se esqueça de criar o requirements.txt

-   **NÃO** se esqueça de criar o .gitignore

-   Esquecer de retornar None caso args > kwargs na `dictionary_creator`
